Asset Properties

An asset is the specific position a portfolio has in an instrument. Instrument properties are independent of portfolio, whereas asset properties are specifc to the portfolio

Property Name Applies To Description Data type Valid values & notes
AssetId All Unique ID of the asset String e.g. ISIN + Market for listed instruments, Alternative Instrument Identifier for OTCs.
Must be unique for every asset in the portfolio and consistent across NAV dates
AssetName All Name of the asset String
HoldingType All Direct or Indirect holding String Only used for display purposes / in forms
Valid values: Direct, Indirect.
Used in TR1 forms
IsDeltaHedge Equity If a short equity position is used for delta hedging purposes of long convertible bonds Boolean Defaults to false if not provided.
ManagedBy All The manager managing the position String Only used for display purposes / in forms
Used in document generation where the manager of individual assets must be disclosed, e.g. 13F
MarketValue All Market value of a position in portfolio currency Decimal If the market value can't be provided in portfolio currency, it should be provided in instrument currency (property MarketValueInInstrumentCurrency). For derivatives the market value should represent the equivalent value of the underlying asset as opposed to the PnL of the contract.
MarketValueInInstrumentCurrency All Market value of a position in instrument currency Decimal Should only be provided if the market value can't be provided in portfolio currency (property MarketValue). For derivatives the market value should represent the equivalent value of the underlying asset as opposed to the PnL of the contract.
Quantity All Quantity of the position Decimal For credit default swaps (CDS), use a negative quantity for protection buys (which is an effective short position) and positive quantity for protection sells (which is an effective long position)

Instrument Properties

An instrument is the portfolio independent information

Property Name Applies To Description Data type Valid values & notes
AssetClass All Asset class (e.g. Equity, Bond, CFD etc.) String Valid values: ADR, Bond, CDS, CFD, Convertible, Equity, Future, Index, Option, Rights, StructuredProduct, Swap, Unit, Warrant.
To ignore this asset during checking, set to 'Ignore'
AssetClassCustomer All Customer defined asset class String Only used for display purposes / in forms
Optional field to be populated with a customer's own internal asset class specification as reference
CallOrPut Warrant, Option If an option / warrant is call or put String Valid values: Call, Put.
ClassDebtOutstanding Convertible Amount of outstanding debt for that class of convertible Decimal Only required for UK takeover calculations for convertibles
ClassSharesOutstanding ADR, Equity, Unit Total amount of shares in circulation for this share class Decimal For Unit funds structured as ETFs this should reflect the number of units issued.
ContractSize CFD, Future, Option, Warrant Contract size of a derivative. If relevant, this should be in instrument currency Decimal e.g. 1000 would mean every option represents 1000 of the underlying.
ConversionRatio ADR, Convertible, Rights Convertible: the number of equities one convertible converts into;
DR: the number of underlying shares represented in one DR;
Right: the number of underlying securities that one right gives entitlement to.		 Decimal
CountryOfIncorporation Equity, Unit Country the issuer is incorporated in String Valid values: ISO 3166-1 Alpha 2 country codes.
CountryOfIssue Bond, Equity Country the stock was issued String Valid values: ISO 3166-1 Alpha 2 country codes.
CUSIP ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Swap, Unit, Warrant 9-digit CUSIP String Only used for US 13F regulation (to match the SEC list). For assets with no CUSIP (e.g. those not traded in the US), default to: NONE
Delta Convertible, Option, Warrant Delta of the asset (price sensitivity with relation to the underlying) Decimal
DRType ADR Type of depositary receipt String Valid values: NVDR, P-Note, Sponsored, Unsponsored.
Required for: Major Thailand
DurationAdjustment Bond Modified duration of a bond Decimal This is used for calculating the appropriate exposure for EU Sovereign Debt Short Selling Regulation.
ETFStructure Unit How the ETF is structured. String Valid values: ETN, Grantor, Open, Partnership, UIT.
ExercisePrice CFD, Future, Option Exercise / strike price Decimal Only used for display purposes / in forms
Used in TR1 forms
ExerciseStyle Option, Warrant Exercise style of options and warrants String Valid values: American, Bermudan, European.
Defaults to American if not provided
FaceValue Bond, Convertible Denomination of a debt instrument at issuance Decimal Defaults to Nominal if not provided.
HomeMemberState Bond, Convertible, Equity, Unit Chosen home member state of the issuer String If not provided, Rapptr tries to look it up in the ESMA list
InControlOfConversion Convertible, Warrant If the holder has control over the conditions of conversion (for convertibles) or exercise (for warrants) Boolean Some countries require convertibles & warrants to be counted if the conditions for conversion are within the control of the parties (e.g. UK)
InstrumentCurrency All
Index, StructuredProduct 		Currency the instrument is denominated in String Valid values: ISO 4217 currency codes.
InstrumentId All Identifier to link assets to instruments and instruments to Underlying instruments String Must be unique for every instrument in one upload
InstrumentName All Instrument name String
IsCashSettled CFD, Future, Option, Swap, Warrant If the derivative is cash settled (as opposed to requiring physical delivery) Boolean Most countries capture physically settled derivatives, many (but less) are interested in cash settled derivatives.
IsCoveredConvertible Convertible If the convertible is covered/on issued shares Boolean
IsCoveredWarrant Warrant If the warrant is covered Boolean
IsETF Unit If a unit fund is exchange traded Boolean
IsGovernmentBacked Bond If the bond is government backed or guaranteed Boolean
ISIN ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Swap, Unit, Warrant ISIN of the instrument String Used to look-up Instrument in many lists such as EU Short Selling Exempt, short selling black & white lists, Takeover lists etc.
IsSubNational Bond If the bond is issued by a country or a state (True = By a state, False = by a country), for sovereign debt rules Boolean Only required if you can't match the sovereign issuer name used in Issuer to ESMA's list of sovereign issuers
IssuerId ADR, Bond, Convertible, Equity, Unit, Warrant Unique identifier for the issuer of the instrument String Defaults to Issuer if not provided.
This allows you to link assets of the same issuer together. If you don't have an Id, the Name can be used, but may not 100% accurate. You could use the issuer's LEI here.
For government bonds, insert the exact sovereign issuer name from ESMA's list of sovereign issuers, if you want Rapptr to lookup the TotalDebtOutstanding automatically.
For Unit funds structured as an ETF, this should be the ETF itself.
IssuerName ADR, Bond, Convertible, Equity, Unit, Warrant Name of the issuer of the instrument String Not needed if you're using the name as the IssuerId. For Unit fund structured as ETFs, the IssuerName should be that of the ETFs it self.
LocalTicker All Local stock ticker String Only used for display purposes / in forms
LotSize ADR, Convertible, Equity Number of shares in each lot or trading unit Decimal Required for: Short selling in Japan
Market ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Rights, Swap, Unit, Warrant Market identification code (MIC) for the market this instrument is traded on String ISO 10383 code (MIC). Used as a lookup to calculate listing country and also for certain rules to see if market is in a list of specific markets. For securities with no market, use value: XXXX
MarketCap Equity Monetary value of the registered share capital in instrument currency Decimal Required for: Phillipines & Czech Republic
MarketsListedIn ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Rights, Swap, Unit, Warrant List of market identification codes (MICs) where the instrument is listed List ISO 10383 code (MIC). For securities with no market, use value: XXXX
MaturityDate Convertible, Future, Option, Swap, Warrant Convertible, Right, Warrant, Option: Expiry date
Physically settled derivative When the securities are to be delivered
Bond, Deposit: Maturity Date		 Date Necessary for USA & Canada to determine if expiry within next 60 days
NextExerciseDate Option Next date where an option is exercisable. For American options: Maturity date Date Defaults to MaturityDate if not provided.
Notional CDS, Future, Option, Swap The reference value of the derivative instrument (in instrument currency) in terms of the underlying debt instrument. Applies to derivatives on bonds or bond indices. This may be the face value or a multiple of the face value of underlying debt. Decimal
ParValue Equity Price of an equity security when originally issued and bearing no relation to the market price. Decimal Required for: Netherlands, Denmark, Egypt (Nominal issued capital regulation)
Price All Price of the instrument in instrument currency Decimal
TitleOfClass All Title of the class of the security (e.g. 1p ORD). Used in disclosure forms String Only used for display purposes / in forms
TotalDebtOutstanding Bond, Convertible Total amount of debt outstanding of an issuer in instrument currency, for sovereign debt rules Decimal Only required if you can't match the sovereign issuer name used in Issuer to ESMA's list of sovereign issuers
TotalIssuedNominalCapital Equity Total amount of issued nominal share capital across all share classes in instrument currency Decimal Required for: Netherlands, Denmark, Egypt (Nominal issued capital regulation)
TotalSharesOutstanding Equity, Unit Total amount of shares in circulation for this issuer (all share classes) Decimal For Unit funds structured as ETFs this should reflect the number of units issued.
TotalVotingRights Equity, Unit Total amount of voting rights in circulation Decimal Defaults to TotalVotingShares if not provided.
TotalVotingShares Equity, Unit Total amount of outstanding shares with voting rights attached in circulation Decimal Required for: Kenya, Kuwait, Malaysia, Thailand
VotesPerShare Equity, Unit Number of votes attached to a single share Decimal
