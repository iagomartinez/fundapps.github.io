Asset Properties

An asset is the specific position a portfolio has in an instrument. Instrument properties are independent of portfolio, whereas asset properties are specifc to the portfolio

Property Name Applies To Description Data type Valid values & notes
AssetId All Unique ID of the asset String e.g. ISIN + Market for listed instruments, Alternative Instrument Identifier for OTCs.
Must be unique for every asset in the portfolio and consistent across NAV dates
AssetName All Name of the asset String
HoldingType All Direct or Indirect holding. Used in TR1 forms to show if holding is direct or indirect String Only used in disclosure forms.
Valid values: Direct, Indirect.
IsDeltaHedge Equity If a short equity position is used for delta hedging purposes of long convertible bonds Boolean Defaults to false if not provided.
ManagedBy All The manager managing the position String Only used in disclosure forms.
MarketValue All Market value of a position in portfolio currency Decimal If the market value can't be provided in portfolio currency, it should be provided in instrument currency (property MarketValueInInstrumentCurrency). For derivatives the market value should represent the equivalent value of the underlying asset as opposed to the PnL of the contract.
MarketValueInInstrumentCurrency All Market value of a position in instrument currency Decimal Should only be provided if the market value can't be provided in portfolio currency (property MarketValue). For derivatives the market value should represent the equivalent value of the underlying asset as opposed to the PnL of the contract.
Quantity All Quantity of the position Decimal

Instrument Properties

An instrument is the portfolio independent information

Property Name Applies To Description Data type Valid values & notes
AssetClass All Asset class (e.g. Equity, Bond, CFD etc.) String Valid values: ADR, Bond, CDS, CFD, Convertible, Equity, Future, Index, Option, Rights, StructuredProduct, Swap, Unit, Warrant.
To ignore this asset during checking, set to 'Ignore'
AssetClassCustomer All Customer defined asset class String Only used in disclosure forms.
CallOrPut Warrant, Option If an option / warrant is Call or Put String Valid values: Call, Put.
ClassDebtOutstanding Convertible The amount of outstanding Debt for that class of convertible Decimal Only required for UK takeover calculations for convertibles
ClassSharesOutstanding ADR, Equity, Unit Total amount of shares in circulation for this share class Decimal
ContractSize CFD, Future, Option, Warrant Contract size of a derivative e.g. 1000 would mean every option represents 1000 of the underlying. (If relevant, this should be in Instrument currency) Decimal
ConversionRatio ADR, Convertible, Rights Convertible: the number of equities one convertible converts into;
DR: the number of underlying shares represented in one DR;
Right: the number of underlying securities that one right gives entitlement to.		 Decimal
CountryOfIncorporation Equity, Unit Country the issuer is incorporated in String Valid values: ISO 3166-1 Alpha 2 country codes.
CountryOfIssue Equity Country the stock was issued String Valid values: ISO 3166-1 Alpha 2 country codes.
CUSIP ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Swap, Unit, Warrant 9-digit CUSIP String Only used for US 13F regulation (to match the SEC list). For assets with no CUSIP (e.g. those not traded in the US), default to: NONE
Delta Convertible, Option, Warrant Delta of the asset (price sensitivity with relation to the underlying) Decimal
DRType ADR The valid type of depositary receipt String Valid values: NVDR, P-Note, Sponsored, Unsponsored.
Required for: Major Thailand
DurationAdjustment Bond Modified duration of a bond Decimal This is used for calculating the appropriate exposure for EU Sovereign Debt Short Selling Regulation.
ExercisePrice CFD, Future, Option Exercise / Strike price Decimal Only used in disclosure forms.
ExerciseStyle Option, Warrant The exercise style of options and warrants String Valid values: American, Bermudan, European.
Defaults to American if not provided
HomeMemberState Bond, Convertible, Equity The chosen Home Member State of the issuer String If not provided, Rapptr tries to look it up in the ESMA list
InControlOfConversion Warrant, Convertible True if the holder has control over the conditions of conversion (for convertibles) or exercise (for warrants) Boolean Some countries require convertibles & warrants to be counted if the conditions for conversion are within the control of the parties (e.g. UK)
InstrumentCurrency All
StructuredProduct, Index 		Currency the Instrument is denominated in String Valid values: ISO 4217 currency codes.
InstrumentName All Instrument name String
IsCashSettled CFD, Future, Option, Swap, Warrant If the derivative is cash settled (as opposed to requiring physical delivery) Boolean Most countries capture physically settled derivatives, many (but less) are interested in cash settled derivatives.
IsCoveredConvertible Convertible If the convertible is covered/on issued shares Boolean
IsCoveredWarrant Warrant If the warrant is covered Boolean
IsGovernmentBacked Bond If the bond is government backed or guaranteed Boolean
ISIN ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Swap, Unit, Warrant ISIN of the instrument String Used to look-up Instrument in many lists such as EU Short Selling Exempt, short selling black & white lists, Takeover lists etc.
IsSubNational Bond Whether a Bond is issued by a country or a state (True = By a state, False = by a country), for Sovereign debt rules Boolean Only required if you can't match the Sovereign issuer name used in Issuer to ESMA's list of Sovereign Issuers
IssuerId ADR, Bond, Convertible, Equity, Warrant Unique identifier for the issuer of the Instrument String Defaults to Issuer if not provided.
This allows you to link assets of the same issuer together. If you don't have an Id, the Name can be used, but may not 100% accurate. (You could use the Issuers LEI here)
IssuerName ADR, Bond, Convertible, Equity, Warrant The name of the Issuer of the Instrument String Not needed if you're using the Name as the IssuerId
LocalTicker All Local stock ticker String Only used in disclosure forms.
LotSize ADR, Convertible, Equity Number of Shares in each lot or trading unit Decimal Required for: Short selling in Japan
Market ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Rights, Swap, Unit, Warrant Market identification code for the market this Instrument is traded on String ISO 10383 code (MIC). Used as a lookup to calculate listing country and also for certain rules to see if market is in a list of specific markets. For securities with no market, use value: XXXX
MarketCap Equity Monetary value of the registered share capital in Instrument Currency Decimal Required for: Phillipines & Czech Republic
MarketsListedIn ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Rights, Swap, Unit, Warrant List of markets (MICs) where the issuer is listed List ISO 10383 code (MIC). For securities with no market, use value: XXXX
MaturityDate Convertible, Future, Option, Swap, Warrant Convertible, Right, Warrant: Expiry date (To determine if expiry within next 60 days)
Physically settled derivative When the securities are to be delivered
Bond, Deposit: Maturity Date		 Date Necessary for USA & Canada
NextExerciseDate Option The next date where an option is exercisable. For American options: Maturity date Date Defaults to MaturityDate if not provided.
Nominal Bond, Convertible Total nominal value of a bond or convertible bond position, or the number of bonds held multiplied by the bond's face value Decimal
ParValue Bond, Convertible Price of a security when originally issued and bearing no relation to the market price, also known as par or face value (or the underlying if a derivative/convertible/warrant) in Instrument Currency Decimal Required for: Netherlands, Denmark, Egypt (Nominal issued capital regulation)
Price All Price of the instrument in Instrument Currency Decimal
TotalDebtOutstanding Bond, Convertible Total debt outstanding of an Issuer in Instrument Currency, for Sovereign debt rules Decimal Only required if you can't match the Sovereign issuer name used in Issuer to ESMA's list of Sovereign Issuers
TotalIssuedNominalCapital Equity The total issued nominal share capital across all share classes in Instrument Currency Decimal Required for: Netherlands, Denmark, Egypt (Nominal issued capital regulation)
TotalSharesOutstanding Equity, Unit Total amount of shares in circulation for this issuer (all share classes) Decimal
TotalVotingRights Equity Total amount of voting rights in circulation Decimal Defaults to TotalVotingShares if not provided.
TotalVotingShares Equity Total amount of outstanding shares with voting rights attached in circulation Decimal Required for: Kenya, Kuwait, Malaysia, Thailand
VotesPerShare Equity Number of Votes attached to a single share Decimal
