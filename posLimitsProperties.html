Properties required for Position Limits

The only way to be sure you are providing enough data is to go into Rapptr results and ensure you have no Missing Data - see here.
Rapptr will fail validation if any of the properties marked with are missing.

Asset Properties

An asset is the specific position a portfolio has in an instrument. Instrument properties are independent of portfolio, whereas asset properties are specifc to the portfolio

Property Name Applies To Description Data type Valid values & notes Rules Jurisdictions
AssetId All Unique ID of the asset String e.g. ISIN + Market for listed instruments, Alternative Instrument Identifier for OTCs.
Must be unique for every asset in the portfolio and consistent across NAV dates		 All All
AssetName All Name of the asset String All All
Quantity All Quantity of the position Decimal For credit default swaps (CDS), use a negative quantity for protection buys (which is an effective short position) and positive quantity for protection sells (which is an effective long position) 489 88
