Asset Properties

Property Name Description Data type Possible values & notes
AssetId Unique identifier for all assets (e.g. ISIN + Market for listed instruments, Alternative Instrument Identifier for OTCs). string Must be unique for every asset in the portfolio and consistent across NAVdates
AssetName Name of the asset string No limitations
Quantity The Quantity of a position decimal

Instrument properties

Property Name Applies to Description Data type Possible values & notes
InstrumentName All Instruments A user friendly name for the instrument string  
InstrumentId All Instruments An identifier to link Assets to instruments, and instruments to Underlying instruments string Can be any string, but must be unique for every instrument 
InstrumentCurrency All Instruments Currency the asset is denominated in string ISO 4217 code. E.g. USD, GBP, EUR
AssetClass All Instruments Asset class string ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, Swap, Unit, Warrant, StructuredProduct (includes Baskets), Rights
CallOrPut Warrant,Option If an option / warrant is call or put string Call, Put
ClassDebtOutstanding Convertible,Bond The amount of outstanding debt for that class decimal Only required for UK takeover calculations for convertibles
ClassSharesOutstanding Equity,ADR,Unit Shares in circulation for this Share Class decimal Used in ca. 30 countries
ContractSize Warrant,Option,Future,CFD Contract size e.g. 1000 would mean every option represents 1000 of the underlying. (If relevant, this should be in Instrument currency) decimal Important for calculating equivalent shares for derivatives
ConversionRatio Convertible,ADR,Rights For a Convertible: the number of equity that one convertible converts into For an ADR: the number of underlying shares represented by one DR For a Right: the number of underlying securities that one right gives entitlement to. decimal For calculating equivalent shares for depository receipts, convertibles and rights.
CountryOfIncorporation Equity Country the issuer is incorporated in string ISO 3166-1 alpha-2 code. E.g. IE, US, DE, FR
CountryOfIssue Equity Country the issue is Issued in string ISO 3166-1 alpha-2 code. E.g. IE, US, DE, FR
Delta Warrant,Option,Convertible Delta of the asset (price sensitivity with relation to the underlying) decimal  
ExerciseStyle Warrant,Option The exercise style of an option string Possible values: American, European
HomeMemberState Equity,Convertible,Bond The chosen Home Member State of the issuer as defined by the EU transparency directive. string ISO 3166-1 alpha-2 code. E.g. IE, US, DE, FR

Rapptr will populate this value, if not provided, based on the ESMA list of MiFID "Shares Traded on Regulated Markets" here.
InControlOfConversion Warrant,Option,Convertible If the conversion is in control of both parties boolean Some countries require convertibles & warrants to be counted if the conditions for conversion are within the control of the parties (e.g. UK)
IsCashSettled Swap,Warrant,Option,Future,CFD If the derivative is cash settled (as opposed to requiring physical delivery) boolean Most countries capture physically settled derivatives, many (but less) are interested in cash settled derivatives.
IsCoveredWarrant Warrant If the warrant is covered boolean Some countries require disclosure of covered warrants but not standard warrants, e.g. Germany.
ISIN Swap, Equity, Warrant, Option, Convertible, ADR, Future, Bond, CFD, CDS, Unit ISIN of the instrument string Used to look-up security in many lists such as EU Short Selling Exempt, short selling black & white lists, Takeover lists etc.
IssuerId Equity,Convertible,ADR,Bond An Id for the Issuer of the asset string This allows you to link assets of the same issuer together. If you don't have an Id, the Name can be used, but may not 100% accurate. (You could use the Issuers LEI here)
IssuerName Equity,Convertible,ADR,Bond The name of the Issuer of the asset string (Not needed if you're using the Name as the Id)
Market Swap, Equity, Warrant, Option, Convertible, ADR, Rights, Future, Bond, CFD, CDS, Unit Market identification code for the market this asset is traded on string ISO 10383 code (MIC). Used as a lookup to calculate listing country and also for certain rules to see if market is in a list of specific markets. For securities with no market, use value: XXXX
MarketsListedIn Swap, Equity, Warrant, Option, Convertible, ADR, Rights, Future, Bond, CFD, CDS, Unit A list of all the markets that the security is traded on list ISO 10383 code (MIC). For securities with no market, use value: XXXX
MaturityDate Warrant,Option,Convertible,Future,Swap Expiry date for warrants, convertibles, rights. Necessary for determining if they expire within next 60 days. For physically settled derivatives in USA & Canada, when the securities are to be delivered date ISO 8601 (YYYY-MM-DD)
Price All Instruments Price of the asset in Instrument Currency decimal Ensure if you don't have a value, leave the cell blank, not 0
TotalSharesOutstanding Equity,Unit Total amount of shares in circulation (i.e. from all share classes) of the Issuer decimal Used in many rules
TotalVotingRights Equity Total amount of voting rights in circulation decimal Used in many rules
VotesPerShare Equity Number of voting rights per share decimal Used to calculate equivalent votes
         

Properties only required for a handful of specific countries
CUSIP Swap, Equity, Warrant, Option, Convertible, ADR, Future, Bond, CFD, CDS, Unit CUSIP of the Instrument string Only used for US 13F regulation (to match the SEC list). For assets with no CUSIP (e.g. those not traded in the US), default to: NONE
ParValue Convertible,Bond Price of a security when originally issued and bearing no relation to the market price, also known as par or face value (or the underlying if a derivative/convertible/warrant) in Instrument Currency decimal Required for: Nominal issued capital regulation in: Netherlands, Denmark, Egypt
TotalIssuedNominalCapital Equity The total issued nominal share capital across all share classes in Instrument Currency  decimal Required for: Nominal issued capital regulation in: Netherlands, Denmark, Egypt
TotalVotingShares Equity Total amount of outstanding shares with voting rights attached in circulation decimal Required for: Kenya, Kuwait, Malaysia, Thailand
Nominal Convertible,Bond Total nominal value of a bond or convertible bond position, or the number of bonds held multiplied by the bond's face value decimal Required for: EU Sovereign debt, UK Takeover Panel
MarketCap Equity Monetary value of the registered share capital in Instrument Currency decimal Required for: Phillipines & Czech Republic
LotSize Equity Number of Shares in each lot or trading unit decimal Required for: Short selling in Japan
         

Required for EUSovereign Debt rules
IsGovernmentBacked Bond Is the bond a Sovereign Bond boolean
DurationAdjustment Bond Modified duration of a bond. This is used for calculating the appropriate exposure for EU short selling regulation. decimal
TotalDebtOustanding Bond Debt outstanding of an Issuer in Instrument Currency decimal Only required if you can't match the Sovereign issuer name used in Issuer to ESMA's list of Sovereign Issuers.
IsSubNational Bond Is the bond issued by a state as opposed to a Country boolean Only required if you can't match the Sovereign issuer name used in Issuer to ESMA's list of Sovereign Issuers.
