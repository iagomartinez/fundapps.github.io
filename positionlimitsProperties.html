Properties required for Positionlimits

Rapptr will fail validation if any of the properties marked with are missing.

Asset Properties

An asset is the specific position a portfolio has in an instrument. Instrument properties are independent of portfolio, whereas asset properties are specifc to the portfolio

Property Name Applies To Description Data type Valid values & notes Rules Jurisdictions
AssetId All Unique ID of the asset String e.g. ISIN + Market for listed instruments, Alternative Instrument Identifier for OTCs.
Must be unique for every asset in the portfolio and consistent across NAV dates		 All All
AssetName All Name of the asset String All All
Quantity All Quantity of the position Decimal For credit default swaps (CDS), use a negative quantity for protection buys (which is an effective short position) and positive quantity for protection sells (which is an effective long position) 2 1

Instrument Properties

An instrument is the portfolio independent information

Property Name Applies To Description Data type Valid values & notes Rules Jurisdictions
AssetClass All Asset class (e.g. Equity, Bond, CFD etc.) String Valid values: ADR, Bond, CDS, CFD, Convertible, Equity, Future, Index, Option, PreferredEquity, Rights, StructuredProduct, Swap, Unit, Warrant.
To ignore this asset during checking, set to 'Ignore'		 All All
CallOrPut Warrant, Option If an option / warrant is call or put String Valid values: Call, Put. All All
CommoditySymbol All Exchange-specific commodity symbol (e.g. CL for NYMEX Crude Oil) String 2 1

ConversionRatio ADR, Convertible, PreferredEquity, Rights Convertible: the number of equities one convertible converts into;
DR: the number of underlying shares represented in one DR;
Right: the number of underlying securities that one right gives entitlement to.
Preferred Equity: the number of common shares one convertible preference share converts into.		 Decimal All All
DeliveryMonth All Delivery month of exchange-traded derivatives in ISO8601 YYYY-MM format String 2 1
Delta Convertible, Option, Warrant Delta of the asset (price sensitivity with relation to the underlying) Decimal All All
InstrumentCurrency All
Index, StructuredProduct 		Currency the instrument is denominated in String Valid values: ISO 4217 currency codes. All All
InstrumentId All Identifier to link assets to instruments and instruments to Underlying instruments String Must be unique for every instrument in one upload All All
InstrumentName All Instrument name String All All
Market ADR, Bond, CDS, CFD, Convertible, Equity, Future, Option, PreferredEquity, Rights, Swap, Unit, Warrant Market identification code (MIC) for the market this instrument is traded on String ISO 10383 code (MIC). Used as a lookup to calculate listing country and also for certain rules to see if market is in a list of specific markets.
